    **Step 2: Add `_ssis_tiled_dims_for_transfer(tr)` method.**

    Returns the list of `LayerDim` from `search_space.dims()` that also appear in `self.ssis[tr].get_applicable_temporal_dimensions()`. This determines which SSIS temporal dimensions have variable tile sizes.

    ```python
    def _ssis_tiled_dims_for_transfer(self, tr: TransferNode) -> list[LayerDim]:
        if self.search_space is None:
            return []
        ssis_temporal_dims = set(self.ssis[tr].get_applicable_temporal_dimensions())
        return [d for d in self.search_space.dims() if d in ssis_temporal_dims]
    ```

    **Step 3: Add `_ssis_coefficients_for_transfer(tr)` method.**

    This is the central new method. For a given transfer, it:
    1. Gets tiled dims via `_ssis_tiled_dims_for_transfer(tr)`
    2. If no tiled dims, returns `None` (signals scalar path)
    3. Otherwise, enumerates all joint candidate combinations across tiled dims using `itertools.product`
    4. For each combination, substitutes the candidate T values into the SSIS temporal sizes list (replacing the sizes for tiled dims with T_k = workload_size / (S * tile_k), using `ssis_loop_sizes_for_candidate`)
    5. Calls `reuse_coefficients_for_sizes(substituted_sizes, relevancies)` to get per-stop coefficients
    6. Gets or creates the joint binary variable for this combination (reuse `_joint_binary_for_combo` pattern from Phase 3, but with a new naming scheme for SSIS, e.g. `ssis_jw_...`)
    7. Returns a dict: `{(tensor, stop): {"fires": [(coeff, jw), ...], "size_factor": [(coeff, jw), ...], "tiles_needed": [(coeff, jw), ...], "bds_needed": [(coeff, jw), ...]}}` for all tensors of this transfer and all stop levels.

    Also compute and store max values for big-M: `_ssis_max_coefficients[(t, stop)]` = `{"fires": max_fires, "size_factor": max_sf, "tiles_needed": max_tn, "bds_needed": max_bn}`.

    **IMPORTANT:** The joint candidate enumeration here differs from `_joint_candidates_for_tensor` (Pitfall 6 in RESEARCH.md). `_joint_candidates_for_tensor` filters dims by tensor shape dependency (inter-core tiling). `_ssis_coefficients_for_transfer` filters dims by SSIS temporal loop dependency (intra-core tiling). Use `_ssis_tiled_dims_for_transfer` NOT `_tiled_dims_for_tensor`.
