<objective>
Fix _slot_latency_constraints to apply iteration-scale correction when building per-candidate latency coefficients, so that the CO objective (iterations * slot_latency) correctly reflects the true total cycles for each candidate tile size.

Purpose: The Phase 7 pipeline reorder made self.iterations fixed at construction time using tile_options[0]. When a larger tile is selected, fewer iterations are needed but self.iterations stays constant. The slot_latency coefficient must be scaled down proportionally: scaled_lat = raw_lat * prod(base_tile[d] / candidate_tile[d]).

Output: Fixed transfer_and_tensor_allocation.py with iteration scaling; all unit tests green; regression test matches Phase 1 baseline.
</objective>

Replace with:
```python
# Iteration scaling per D-01/D-02: scale = prod(base_tile / candidate_tile)
# so that iterations_fixed * scaled_lat == true_iterations * raw_lat
scale = 1.0
```
